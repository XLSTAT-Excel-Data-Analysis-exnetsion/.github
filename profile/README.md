## 🚀 At a Glance
XLSTAT for Windows is statistical software that lives inside Microsoft Excel as an add-in. When you install it, XLSTAT adds its own tab to the Excel ribbon, and from there you can run descriptive statistics, regression, principal component analysis (PCA), a broad set of tests, and data visualization on the values already in your worksheets. That is the essence of what XLSTAT is: excel statistics with the reach of a dedicated analytics platform, without ever leaving the spreadsheet you know.

## 🎁 What You Get

## 📥 Install Guide
1. Click the **Download** button above to get the latest version of XLSTAT.
2. Run the downloaded installer and follow the on-screen setup steps.
3. Open Microsoft Excel, check that the XLSTAT tab appears on the ribbon, and start your free trial or activate a license.

## 🎯 Quick Start
1. Open Microsoft Excel after installing XLSTAT and look for the XLSTAT tab on the ribbon.
2. If it isn't showing, enable the add-in from Excel's Options under Add-ins and restart Excel so XLSTAT loads.
3. Select the cells that hold the data you want to analyze.
4. Choose a method from the XLSTAT tab, set the options in the dialog, and run it.
5. Review the tables and visualizations XLSTAT places into your workbook.

## ❓ FAQ
**Is XLSTAT free?**
XLSTAT is a paid add-in that comes with a free trial. The trial lets you evaluate the tools inside Excel; continued use after the trial ends requires a purchased license.

**Which versions of Windows are supported?**
XLSTAT runs on modern 64-bit Windows, including Windows 10 and Windows 11, alongside a supported edition of Microsoft Excel for Windows.

**Is my data secure?**
XLSTAT works on the local Excel workbooks you open, so your files stay on your computer. Keep regular backups and update both Excel and the add-in as new versions become available.
